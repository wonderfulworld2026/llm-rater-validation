
# Evaluating an LLM Rater for Constructed-Response Assessment

> A reproducible evaluation framework for assessing LLM-based scoring using established educational measurement methods.

---

## Abstract

Large language models (LLMs) have demonstrated language understanding capabilities that extend beyond traditional semantic analysis approaches, creating new opportunities for educational assessment. One promising application is the scoring of constructed-response questions, where student responses must be evaluated for their meaning rather than matched to predefined choices. Before LLMs can be considered for educational scoring, their performance should be evaluated against human ratings using rigorous and appropriate educational measurement methods.

This repository presents an empirical evaluation of an **LLM rater** for scoring short constructed responses. Using the publicly available **Mohler Short Answer Dataset**, scores generated by **Claude Haiku 4.5** are compared with reference scores derived from two independent human raters. Agreement between the LLM rater and the human reference scores is evaluated using multiple complementary metrics, including Quadratic Weighted Kappa (QWK), Pearson correlation, Spearman correlation, Exact Agreement, Adjacent Agreement, and Mean Absolute Error (MAE), rather than relying on a single statistic. In addition to the primary evaluation, the repository includes a lexical-overlap baseline, compares zero-shot and few-shot prompting strategies, and examines scoring agreement across responses of different lengths.

Rather than proposing a new scoring model, this project presents a transparent and reproducible workflow for evaluating LLM-based scoring systems using established educational measurement methods.

# Why This Project?

Constructed-response questions provide richer evidence of student understanding than selected-response questions because learners must generate their own responses rather than select from predefined answer choices. However, scoring large numbers of constructed responses manually requires substantial time and expert human judgment.

Recent advances in large language models (LLMs) present new opportunities to support large-scale scoring of constructed responses. Before LLMs can be considered for educational scoring applications, however, their scoring performance should be evaluated against human reference ratings using rigorous educational measurement methods.

The goal of this project is to provide a reproducible workflow for assessing the agreement between LLM-generated scores and human reference ratings using established educational measurement metrics, supporting future research on scalable, reliable LLM-assisted scoring of constructed-response assessments.

---

# Evaluation Workflow

The study consists of four components:

1. **LLM Rater** – Claude Haiku 4.5 scores each response using a researcher-developed scoring rubric.
2. **Lexical-overlap Baseline** – A simple heuristic provides a lightweight automated point of comparison.
3. **Agreement Evaluation** – Machine-generated scores are compared with the dataset's reference scores.
4. **Additional Analyses** – Few-shot prompting and response-length subgroup analyses evaluate scoring robustness.

---

# Dataset

This study uses the publicly available **Mohler Short Answer Dataset**, which was developed using responses from undergraduate students enrolled in an introductory computer science course at the University of North Texas. The dataset contains **79 short constructed-response questions** and **2,273 student responses**, each independently scored by two human raters. As an internal reference point for interpreting agreement on this dataset, the two independent human raters agree with one another at a Pearson correlation of approximately r = 0.586 (Mohler et al., 2011), reflecting the inherent subjectivity of short constructed-response scoring even among expert raters.

To reduce API cost while maintaining a representative evaluation sample, this study sampled **150 student responses** using **stratified random sampling** across the reference score distribution. This sampling strategy preserves the overall score distribution of the full dataset and avoids over-representing the more frequently occurring high-score responses. The same fixed sample was used for all experimental conditions, ensuring fair and reproducible comparisons across the lexical-overlap baseline, zero-shot prompting, and few-shot prompting.

Each response includes:

- Assessment question
- Reference answer
- Student response
- Two independent human ratings on an ordinal scale from **0** (completely incorrect) to **5** (completely correct)
- Reference score (the average of the two human ratings, provided by the dataset)

The reference score serves as the comparison target throughout this study. It is used **only during evaluation** and is **never provided to the LLM rater during scoring**.

# Evaluation Framework

Agreement between the LLM rater and the reference scores is evaluated using multiple complementary measures.

| Metric | Purpose |
|--------|---------|
| **Quadratic Weighted Kappa (QWK)** | Primary agreement statistic for ordinal ratings |
| **Pearson Correlation** | Linear association between scores |
| **Spearman Correlation** | Rank-order association |
| **Exact Agreement** | Percentage of identical scores |
| **Adjacent Agreement** | Percentage of scores within one point |
| **Mean Absolute Error (MAE)** | Average magnitude of scoring differences |

Reporting several metrics provides a more complete view of scoring agreement than any single measure alone.

# Results

The following observations are based on the evaluation results obtained in this study.

- **LLM rater vs. lexical-overlap baseline:** Across all reported evaluation metrics, the LLM rater demonstrated stronger agreement with the human reference scores than the lexical-overlap baseline, suggesting that semantic reasoning provides measurable improvement over simple lexical matching (See Table 1). Examination of the score confusion matrix further indicates that disagreements were rarely severe — the LLM rater seldom differed from the human reference by more than one point — and tended to fall slightly below the human reference on the highest-scoring responses, suggesting a mild, systematic strictness rather than random error.

- **Zero-shot vs. few-shot prompting:** Adding a small number of scored examples to the prompt produced only modest improvements over the zero-shot condition, indicating that the researcher-developed scoring rubric alone provided relatively stable scoring performance (See Table 2). This pattern is consistent with prior work: Henkel et al. (2024), evaluating LLM-based scoring of short answers in a K-12 setting, similarly reported only negligible gains from few-shot over zero-shot prompting (Cohen's κ 0.44 → 0.45 for GPT-3.5 and 0.68 → 0.70 for GPT-4), though that study used a binary correct/incorrect task on a different learner population.

- **Response length:** Scoring agreement varied across response-length groups. As response length increased from short to long, the agreement between the LLM rater and the human reference scores (measured by QWK) generally decreased, suggesting that longer responses present a more challenging scoring task (See Table 3).

# Potential Applications

Although this repository evaluates **Claude Haiku 4.5**, the evaluation workflow is designed to be model-agnostic and can be readily adapted to compare:

- Different LLMs
- Alternative prompting strategies
- Researcher-developed scoring rubrics
- Educational datasets
- Subject domains

The workflow provides a reproducible approach for evaluating LLM-based scoring systems and can support future research on scalable, LLM-assisted scoring of constructed-response assessments.

---

# Limitations

This study has several limitations that should be considered when interpreting the results.

- **Single dataset and domain.** The evaluation is based on the Mohler Short Answer Dataset, which consists of introductory computer science questions. Agreement observed in this study may not generalize to other subject areas, assessment formats, or learner populations.

- **Reference score.** The reference score is the average of two independent human ratings provided by the dataset. Because human ratings are themselves subject to measurement error, the LLM rater can only be evaluated against this reference rather than an absolute "true" score.

- **Single-pass scoring.** Each response was scored once by the LLM. This study does not evaluate the consistency of the LLM rater across repeated scoring sessions (i.e., test–retest reliability).

- **Sample size and score distribution.** The evaluation uses a stratified sample of 150 responses. Although stratified sampling preserves the overall score distribution of the dataset, the dataset is skewed toward high-scoring responses (mean reference score ≈ 4.17), resulting in relatively few responses with mid-range scores (2–3). Consequently, agreement estimates for the middle of the score scale should be interpreted with greater caution than those for the more frequently occurring high scores.

---

# Future Research

Potential directions for extending this work include:

- **LLM rater reliability.** Score each response multiple times to evaluate the self-consistency of the LLM rater (test–retest reliability) in addition to agreement with human reference scores.

- **Fairness analyses.** Examine whether scoring agreement differs across response characteristics that should not influence scoring decisions, such as verbosity, writing style, or lexical variation.

- **Long-response scoring.** Investigate why scoring agreement decreases for longer responses and evaluate whether alternative prompting strategies or scoring rubrics improve agreement.

- **Capability growth measurement.** Extend the framework from scoring individual responses to measuring learner growth over time, enabling evaluation of whether students demonstrate meaningful improvement across repeated assessments.
